# Computer Science & Engineering (AI/ML Specialization)

## Program Overview

**B.Tech in CSE (AI/ML)** is a specialized program at BCREC focusing on Artificial Intelligence and Machine Learning.

## Key Details

| Attribute | Value |
|-----------|-------|
| **Duration** | 4 Years (8 Semesters) |
| **Intake** | 60 Students |
| **Affiliation** | MAKAUT |
| **Accreditation** | NBA |

## Eligibility

- 10+2 with Physics, Chemistry, Mathematics
- Minimum 50% aggregate marks
- Valid WBJEE or JEE Main score

## WBJEE Cutoff (2025)

| Category | Round 1 | Round 2 |
|----------|---------|---------|
| General | 78,000 | 99,000 |

## Placement Statistics

- **Placement Rate**: 90-95%
- **Average Package**: ₹5-7 LPA
- **Highest Package**: ₹15-20 LPA

## Curriculum Overview

### Year 1 (Foundation)
- Engineering Mathematics
- Physics & Chemistry
- C Programming
- Data Structures
- Digital Logic

### Year 2 (AI Basics)
- Database Management Systems
- Operating Systems
- Python Programming
- Statistics & Probability
- Discrete Mathematics

### Year 3 (AI/ML Core)
- Machine Learning
- Deep Learning
- Natural Language Processing
- Computer Vision
- Data Analytics

### Year 4 (Advanced)
- AI Project
- Industrial Training
- Electives in AI/ML
