# Information Technology (IT)

## Program Overview

**B.Tech in Information Technology** at BCREC focuses on software development, database management, and IT systems.

## Key Details

| Attribute | Value |
|-----------|-------|
| **Duration** | 4 Years (8 Semesters) |
| **Intake** | 120 Students |
| **Affiliation** | MAKAUT |
| **Accreditation** | NBA |

## Eligibility

- 10+2 with Physics, Chemistry, Mathematics
- Minimum 50% aggregate marks
- Valid WBJEE or JEE Main score

## WBJEE Cutoff (2025)

| Category | Round 1 | Round 2 |
|----------|---------|---------|
| General | 7,500 | 53,000 |

## Placement Statistics

- **Placement Rate**: 80-85%
- **Average Package**: ₹3.5-4.5 LPA
- **Highest Package**: ₹8 LPA

## Curriculum Overview

### Year 1 (Foundation)
- Engineering Mathematics
- Physics & Chemistry
- C Programming
- Data Structures
- Digital Logic

### Year 2 (IT Core)
- Database Management Systems
- Operating Systems
- Computer Networks
- Software Engineering
- Web Technology

### Year 3 (Advanced)
- Cloud Computing
- Cyber Security
- Mobile App Development
- Data Science Basics
- IT Infrastructure

### Year 4 (Specialization)
- Capstone Project
- Industrial Training
- Electives
