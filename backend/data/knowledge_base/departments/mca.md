# Master of Computer Applications (MCA)

## Program Overview

**MCA at BCREC** is a 3-year program focusing on software development, databases, and computer applications.

## Key Details

| Attribute | Value |
|-----------|-------|
| **Duration** | 3 Years (6 Semesters) |
| **Intake** | 60 Students |
| **Affiliation** | MAKAUT |
| **Accreditation** | NBA |

## Eligibility

- Bachelor's degree with Mathematics at 10+2 level
- Minimum 50% marks
- Valid WB-JECA score

## Entrance Exam

- WB-JECA (West Bengal Electronic Common Entrance Examination)

## Fee Structure

- **Total Fees**: ₹2.17 Lakhs

## Placement Statistics

- **Placement Rate**: 70-80%
- **Average Package**: ₹3.5-4.5 LPA

## Curriculum Overview

### Year 1
- Programming in C
- Data Structures
- Computer Architecture
- Discrete Mathematics
- Operating Systems

### Year 2
- Database Management Systems
- Object Oriented Programming (Java/C++)
- Software Engineering
- Computer Networks
- Web Technology

### Year 3
- Cloud Computing
- Mobile Application Development
- Artificial Intelligence
- Project Work
- Summer Training

## Career Opportunities

- Software Developer
- Web Developer
- Database Administrator
- System Analyst
- Mobile App Developer
- QA Engineer

TCS, Infosys, Wipro, Cognizant, Capgemini, IBM, Accenture, Tech Mahindra
